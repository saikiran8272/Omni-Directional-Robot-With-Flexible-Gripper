# Omni-Directional-Robot-With-Flexible-Gripper
Design and fabrication of an Omni-Directional Robot with flexible gripper mechanism for efficient industrial pick-and-place operations using embedded systems, robotics, and 3D printing.

## Project Overview
This project focuses on the Design and fabrication of an Omni-Directional Robot (ODR) integrated with a flexible gripper mechanism for efficient industrial pick-and-place applications.

The robot is capable of movement in all directions without changing orientation using omni wheels and embedded motor control systems.

The project combines:
- Mechanical Design
- Embedded Systems
- Robotics
- 3D Printing
- Industrial Automation
  
# Objective
To design and fabricate an Omni-Directional Robot (ODR) with gripper mechanism that can easily move in all directions for efficient pick-and-place operations through a flexible gripper capable of handling intricate shaped industrial objects.

# Key Features
- Omni-Directional Movement
- Flexible Robotic Gripper
- Arduino-based motor control

# Working Principle

The omni wheels allow the robot to move in forward, backward, sideways, and diagonal directions. Motor control is achieved using Arduino and Motor driver modules. The gripper mechanism is controlled using servo motors for efficient object handling.

# CAD Design
CAD models were designed using SolidWorks for:
- Chassis Assembly
- Gripper Housing
- Robotics ARM

# 3D Printing
All CAD models were fabricated using Fused Deposition Modelling(FDM):

- Printer Used - Creality 3D

- Slicing Software - Prusa Slicer
